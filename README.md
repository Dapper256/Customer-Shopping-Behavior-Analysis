Project Overview
This project delivers a comprehensive end-to-end customer shopping behavior analysis pipeline built to simulate a production-grade business intelligence environment. Using Python, PostgreSQL, SQL, and Power BI, the analysis transforms raw transactional retail data into strategic business insights that support executive decision-making across customer retention, revenue optimization, pricing strategy, and operational performance.
Analyzing 3,900+ customer transactions across 18 behavioral, demographic, and commercial dimensions, this project demonstrates how modern analytics can uncover hidden revenue drivers, identify high-value customer segments, optimize discount governance, and improve product marketing strategies.
